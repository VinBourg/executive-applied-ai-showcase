# Agentic Knowledge Routing

A compact agentic workflow demo that turns a business request into a structured execution plan with clear agents, tools, checks and delivery logic.

## Why it matters

This example demonstrates:
- agentic task decomposition,
- routing between retrieval, validation and synthesis steps,
- business-oriented orchestration logic,
- explainable workflow planning.

## Business relevance

- AI task routing,
- knowledge work automation,
- agent planning logic,
- structured business assistance.

## Files

- `app.py`
  Builds execution plans and writes a readable report.
- `task_briefs.json`
  Example requests aligned with operations, fraud and automation use cases.
- `execution_plan_report.md`
  Generated output showing the structure of the agentic plan.

## Run

```bash
python3 app.py
```

## What a recruiter should see quickly

This folder shows that agentic workflow design can be:
- explicit rather than vague,
- tied to deliverables and deadlines,
- controlled through validation checks,
- readable for business stakeholders.
