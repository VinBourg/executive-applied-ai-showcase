# AI Readiness Value Risk Assessment

Consulting-style assessment for selecting agentic AI opportunities by business value, data readiness, adoption feasibility, risk and integration complexity.

## Business Use Case

Many AI initiatives fail because the use case is selected too quickly. This item shows a practical way to compare candidates and decide what should be piloted now, what needs controls and what should stay in discovery.

## What It Demonstrates

- Value/risk/readiness scoring for AI and agentic automation opportunities.
- OpenClaw and n8n positioned only when the operating model requires controlled agent reasoning and workflow handoff.
- A pilot recommendation layer that remains business-readable.
- Control matrix and 30-day roadmap useful for a client steering discussion.

## Outputs

- `readiness_assessment_report.md`
- `use_case_portfolio.csv`
- `control_matrix.csv`
- `thirty_day_pilot_roadmap.md`
