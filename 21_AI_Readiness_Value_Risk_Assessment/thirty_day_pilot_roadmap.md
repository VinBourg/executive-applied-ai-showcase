# 30-Day Pilot Roadmap - AI Readiness Assessment

## Week 1 - Framing

- Confirm owner, users, business decision and expected output.
- Validate data sources, access constraints and success metrics.

## Week 2 - Prototype

- Build first controlled prototype for `AI SQL portfolio copilot` using OpenClaw analyst agent + SQL review workflow.
- Build first controlled prototype for `Claims document intake agent` using OpenClaw reasoning layer + n8n approval workflow.
- Build first controlled prototype for `KPI narrative generation` using n8n scheduled reporting + LLM narrative review.

## Week 3 - Evaluation

- Run user review, output scoring, risk review and adoption feedback.

## Week 4 - Decision

- Decide whether to stop, extend, industrialize or reframe the pilot.
- Produce executive memo, action queue and ownership map.
