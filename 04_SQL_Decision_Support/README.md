# SQL Decision Support

A small SQLite example showing how SQL can be used to produce business-oriented decision-support outputs.

Use case:
- SQL data mart logic
- segment performance review
- account prioritization

Generated outputs:
- `decision_support_report.md`
- `account_priority_queue.csv`

Operational value:
- gives a team a concrete follow-up queue instead of raw rows,
- turns SQL into a business review tool,
- supports account-level prioritization with simple, readable logic.
