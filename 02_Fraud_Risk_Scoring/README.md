# Fraud Risk Scoring

A compact Python demo of supervised learning for fraud-risk scoring.

Use case:
- interpretable risk scoring
- fraud analytics proof of concept
- predictive analytics showcase

Generated outputs:
- `fraud_scoring_report.md`
- `fraud_decisions.csv`

Operational value:
- turns raw scoring into an analyst-readable decision layer,
- makes risk drivers explicit rather than black-box,
- provides a direct review or block recommendation.
