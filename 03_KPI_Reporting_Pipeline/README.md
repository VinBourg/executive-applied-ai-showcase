# KPI Reporting Pipeline

A Python reporting pipeline that transforms raw support operations data into decision-ready KPI outputs.

## Why it matters

This example demonstrates:
- reporting automation
- KPI production for decision support
- Power BI-ready exports
- a path from pipeline logic to dashboard presentation

## Files

- `app.py`
  Local pipeline runner.
- `metrics.py`
  Shared logic for KPI computation and dashboard generation.
- `dashboard.html`
  Executive-style HTML dashboard generated by the pipeline.
- `streamlit_app.py`
  Optional interactive dashboard version for GitHub demonstration.
- `tickets.csv`
  Small source dataset.

## Run the local pipeline

```bash
python3 app.py
```

This generates:
- `kpi_summary.json`
- `kpi_by_category.csv`
- `kpi_by_day.csv`
- `dashboard.html`

## Optional Streamlit version

```bash
streamlit run streamlit_app.py
```
